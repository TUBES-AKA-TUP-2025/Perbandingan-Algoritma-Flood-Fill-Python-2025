# Perbandingan-Algoritma-Flood-Fill-Python-2025
Perbandingan algoritma Flood Fill: Rekursif vs Iteratif. Menampilkan analisis kompleksitas algoritma, benchmark running time, dan visualisasi grafik menggunakan Python untuk mendemonstrasikan efisiensi memori dan stabilitas sistem pada berbagai ukuran grid.


Analisis Kompleksitas Algoritma Flood Fill (Paint Bucket): Perbandingan Pendekatan Rekursif dan Iteratif Berbasis Stack	"   Tugas besar ini bertujuan menganalisis algoritma Flood Fill yang umum ditemukan pada aplikasi pengolah gambar (misalnya, Paint Bucket Tool). Studi kasusnya adalah mewarnai sebuah area tertutup pada sebuah grid/matriks 2D. Algoritma ini akan diimplementasikan dalam dua versi:    1) Versi Rekursif: Sebuah fungsi yang memanggil dirinya sendiri untuk mewarnai piksel tetangga (atas, bawah, kiri, kanan). Analisis kompleksitasnya akan menghasilkan relasi rekurensi yang dapat diselesaikan.     2) Versi Iteratif: Menggunakan struktur data Stack (tumpukan) dan sebuah perulangan (while loop) untuk mengelola piksel mana yang harus diwarnai selanjutnya, meniru cara kerja pemanggilan fungsi rekursif. Analisisnya dilakukan dengan menghitung jumlah operasi di dalam perulangan (analisis iteratif).     Perbandingan akan berfokus pada kompleksitas waktu (O,Ω,Θ), penggunaan memori (risiko stack overflow pada versi rekursif), dan running time aktual untuk berbagai ukuran grid."
